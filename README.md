## JavaScript Practice Problems

### Problem 1: Basic Arithmetic

Write a function that takes two numbers and returns their sum, difference, product, and division.

- **Example:**
- **Input:** `5, 4`
- **Output:** `[9, 1, 20, 1.25]`

### Problem 2: Even or Odd

Given a number, check whether it is even or odd.

- **Example:**
- **Input:** `7`
- **Output:** `"Odd"`

### Problem 3: Find the Largest Number

Write a function to find the largest number in an array.

- **Example:**
- **Input:** `[1, 5, 3, 9, 2]`
- **Output:** `9`

### Problem 4: Count Vowels

Given a string, count how many vowels (`a, e, i, o, u`) it contains.

- **Example:**
- **Input:** `"javascript"`
- **Output:** `3`

### Problem 5: Reverse a String

Write a program to reverse a string.

- **Example:**
- **Input:** `"hello"`
- **Output:** `"olleh"`

### Problem 6: Array Total Sum

Given an array of numbers, calculate the total sum of all elements.

- **Example:**
- **Input:** `[10, 20, 30]`
- **Output:** `60`

### Problem 7: Prime Number Check

Write a function that checks if a number is prime (only divisible by 1 and itself).

- **Example:**
- **Input:** `11`
- **Output:** `true`

### Problem 8: Filter Positive Numbers

Given an array, create a new array containing only the positive numbers.

- **Example:**
- **Input:** `[-2, 5, 0, -8, 12]`
- **Output:** `[5, 12]`

### Problem 9: Capitalize Words

Write a function to capitalize the first letter of each word in a sentence.

- **Example:**
- **Input:** `"hello world"`
- **Output:** `"Hello World"`

### Problem 10: FizzBuzz

Given a number `n`, print numbers from 1 to `n`. Replace multiples of 3 with "Fizz" and multiples of 5 with "Buzz".

- **Example:**
- **Input:** `5`
- **Output:** `1, 2, "Fizz", 4, "Buzz"`

### Problem 11: Count Occurrences

Write a function that counts how many times a specific value appears in an array.

- **Example:**
- **Input:** `([1, 2, 1, 3, 1], 1)`
- **Output:** `3`

### Problem 12: Array Equality

Check if two arrays are equal (same values in the same order).

- **Example:**
- **Input:** `[1, 2], [1, 2]`
- **Output:** `true`

### Problem 13: Swap Variables

Swap two variables without using a third (temporary) variable.

- **Example:**
- **Input:** `a = 5, b = 10`
- **Output:** `a = 10, b = 5`

### Problem 14: Find Shortest Word

Given an array of strings, find the shortest word.

- **Example:**
- **Input:** `["apple", "pie", "banana"]`
- **Output:** `"pie"`

### Problem 15: Temperature Converter

Write a function that converts Celsius to Fahrenheit.

> The formula is: F = (C Ã— 9/5) + 32

- **Example:**
- **Input:** `0`
- **Output:** `32`
